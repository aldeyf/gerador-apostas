Apresentação

Loterias Caixote - O melhor lugar para fazer suas apostas!

Este é Projeto de Gerador de Apostas, uma aplicação que utiliza HTML, CSS e JavaScript. A aplicação é capaz de gerar uma aposta mínima de cada um dos jogos das loterias (utilizando as mesmas regras da Mega-Sena; Quina; Lotofácil e Lotomania), mas com uma pitada de humor, nas Loterias Caixote utilizamos outra nomenclatura apenas para descontrair.

Desde 2018 a Loterias Caixote está disponível para apostas. Em nosso ambiente você terá a oportunidade de ficar rico sem estudar ou trabalhar. Olha que maravilha!!! Então corra logo para nossa página "Gerador" para escolher em qual jogo vai querer apostar. Basta clicar lá em cima no menu principal "Gerador" e ser feliz.

Sobre o projeto
