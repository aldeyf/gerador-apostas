Lotomané


A Lotomané é fácil de jogar e de ganhar para você que é mané. Para quem joga nesta loteria, serão gerados 50 números dentre 100. São sorteados 20 números, e os bilhetes premiados são aqueles em que os jogadores acertam de 20 a 16 números. Para gerar outro número basta clicar em Gerar nova aposta.

Sobre o projeto

Projeto de Gerador de Apostas, a aplicação utiliza apenas HTML, CSS e JavaScript.

Como a intenção era utilizar javascript apenas, não foi feito uso de bibliotecas como jquery, nem uso de nenhum outro framework.
